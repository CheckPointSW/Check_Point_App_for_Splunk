Check Point brings you an advanced and real-time threat analysis and reporting tool for Splunk. The Check Point App for Splunk allows you to respond to security risks immediately and gain network true insights.
Collect and analyze millions of logs from all Check Point technologies and platforms across Networks, Cloud, Endpoints and Mobile.
This app uses Check Point’s Log Exporter to seamlessly send logs out of your Check Point log server into your Splunk server – for more details, see Check Point SK122323.

Main Features include:
    * Compatibility for Common Information Model (CIM)
    * Compatibility with Splunk Enterprise Security
    * Integrated with Check Point SmartEvent dashboards (e.g., General Overview, Threat Prevention and the new Cyber Attack View)
